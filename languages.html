Define a new language

You can easily define your own language if you can't find what you want in the list of supported languages.

{% highlight javascript %} // define a language numbro.language('fr-BE', { delimiters: { thousands: ' ', decimal: ',' }, abbreviations: { thousand: 'k', million: 'm', billion: 'b', trillion: 't' }, ordinal : function (number) { return number === 1 ? 'er' : 'ème'; }, currency: { symbol: '€', position: 'postfix' }, defaults: { currencyFormat: ',0000 a' }, formats: { fourDigits: '0000 a', fullWithTwoDecimals: ',0.00 $', fullWithTwoDecimalsNoCurrency: ',0.00', fullWithNoDecimals: ',0 $' } }); {% endhighlight %} If you lack a language, please help us to improve numbro by submitting a pull request.

Change language

To change the language to use, you only need to use the language function with the culture code of the language needed.

{% highlight javascript %} // switch between languages numbro.culture('fr-BE'); {% endhighlight %}
